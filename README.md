Add-on for pajic's Accessorify made by GK1H.
<h1>Features</h1>
<h2>Both Fabric & NeoForge</h2>
Compat with Totem of Neverdying from <a href="https://github.com/SLH335/netherite-extras">Netherite Extras</a>.
<h2>NeoForge</h2>
Compat with <a href="https://github.com/Aton-Kish/reinforced-shulker-boxes">Reinforced Shulker Boxes</a>. (Requires <a href="https://github.com/Gameking1happy-Development/GK1H-Tags">GK1H Tags</a>.)
